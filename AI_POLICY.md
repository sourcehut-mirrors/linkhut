# AI Policy

## This project

Generative AI tools have been used in the development of linkhut, primarily for research, learning, and code review,
with some code generation. The motivation for the use of these tools was for me to explore the landscape and determine
their utility on a project that I was deeply familiar with. Although very little AI-generated code was produced, the
output of these tools did influence my thinking and approach to various non-coding tasks.

I want to be upfront about that, because I know it matters to some people. If you've taken a harder line on AI-generated
code and this is a deal breaker, I respect that.

That being said. Moving forward, I won't be using generative AI tools to work on linkhut anymore. If I feel the need to
revisit the landscape of this technology in the future, I won't be doing so on this project.

## Contributing

Henceforth, use of AI tools for contributing to linkhut is banned.
